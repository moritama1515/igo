Igo
====
簡単囲碁プログラムです．  
ランダムに手を発生して打つところまで実装してあります．  
続きの開発をPrivate Repositoryに移行しました．

## Description
boardは人対人用のプログラムです．
tamagoは対人プログラム．（ランダムに打つだけ）

## Usage
x -> (Input from 1 to 19)  
y -> (Input from 1 to 19)

If you want to exit the program , please enter 0; 

xとyの入力を求められるので順に入力してください．
パスは20，終了は0です．
